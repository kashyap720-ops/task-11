# Thala 7 Sneaker Store

## Description
Thala 7 Sneaker Store is a responsive e-commerce website built using HTML and CSS. It showcases premium sneakers with a modern user interface, shopping cart, and checkout page.

## Features

- Responsive Landing Page
- Product Details Page
- Shopping Cart
- Checkout / Payment Page
- Modern Navigation Bar
- Search Box
- Responsive Design
- Hover Effects
- CSS Animations
- Flexbox & Grid Layout

## Technologies Used

- HTML5
- CSS3
- Font Awesome

## Project Structure

├── index.html
├── products.html
├── cart.html
├── payment.html
├── style.css
└── README.md

## How to Run

1. Download or clone the project.
2. Open `index.html` in any web browser.
3. Navigate through the pages using the navbar.
